# Deployment Files

This directory contains all files needed for deploying AI Newsroom to production.

## Files

- **COMPLETE_DEPLOYMENT_GUIDE.md** - Full deployment guide with step-by-step instructions, troubleshooting, and maintenance
- **QUICK_REFERENCE.md** - Quick reference card for common commands and operations

- **nginx.conf** - Nginx web server configuration template
- **ainews-backend.service** - Systemd service file for backend
